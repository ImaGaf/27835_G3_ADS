# Línea Base LBA - Análisis

Esta carpeta contiene los documentos y diagramas relacionados con la fase de análisis del proyecto **Pago Seguro AGROTAC**.

## Contenido principal:
* **Diagramas de actividades:** Flujos de trabajo del sistema (gestión de pagos, usuarios y notificaciones).
Los diagramas se han organizado por tipo de usuario con el objetivo de facilitar la comprensión de los procesos y responsabilidades de cada rol dentro de la plataforma (V1.1)
* **Documentación de análisis funcional:** Desglose de procesos clave para la modernización de la gestión de comprobantes.
