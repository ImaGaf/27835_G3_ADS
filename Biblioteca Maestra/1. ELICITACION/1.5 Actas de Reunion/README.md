# Actas de Reunión - Proyecto PagoSeguro

Esta carpeta contiene la documentación oficial y técnica relacionada con las reuniones y revisiones de requisitos del sistema PagoSeguro, incluyendo las últimas actualizaciones aprobadas por el comité del proyecto.

## Contenido Principal

- **Documento SRS (Software Requirements Specification):** Especificación completa de requisitos funcionales y no funcionales.
- **Acta de Revisión G3:** Documento de respaldo que formaliza las mejoras en seguridad y protección de datos.

## Fechas Clave

- **Última Revisión:** 04 de diciembre de 2025
- **Plazo de Actualización:** Entrega de ajustes de seguridad en un máximo de 5 días laborales
- **Fecha de Inicio (Carpeta):** 11 de diciembre de 2025

## Enfoque Actual de Seguridad

Tras la sesión de análisis, se han fortalecido los siguientes apartados para garantizar la robustez del sistema:

- **Niveles de Acceso:** Definición precisa según el rol del usuario
- **Autenticación:** Descripción ampliada de mecanismos de validación
- **Auditoría:** Implementación de procedimientos estrictos de trazabilidad
- **Recuperación:** Políticas claras ante incidentes o brechas de seguridad
